# 🎮 NitenVerse

> Uma jornada interativa pela história da Nintendo, seus consoles icônicos e jogos que marcaram gerações.

---

## 📌 Sobre o Projeto

O **NitenVerse** é um site desenvolvido com o objetivo de apresentar a evolução da Nintendo ao longo dos anos, destacando:

- 🕹️ Consoles históricos  
- 🎮 Jogos icônicos  
- 📅 Linha do tempo das gerações  
- 👨‍💻 Páginas individuais com detalhes completos  
- 📬 Página de contatos dos criadores  

O projeto foi criado com foco em design moderno, organização visual e experiência interativa para os fãs da Nintendo.

---

## 🚀 Tecnologias Utilizadas

- ✅ HTML5  
- ✅ CSS3  
- ✅ Layout Responsivo  
- ✅ Estrutura organizada por páginas individuais  

---
## 🎨 Funcionalidades

- 🎮 Página de Jogos com cards informativos  
- 🕹️ Página de Consoles em formato de linha do tempo  
- 📬 Página de Contatos com foto, nome e email dos criadores  
- 🎨 Hero sections personalizados para cada aba  

---

## 🌟 Objetivo

O projeto foi desenvolvido para:

- Praticar desenvolvimento web  
- Trabalhar estruturação de layout  
- Explorar design inspirado no universo Nintendo  
- Criar uma experiência temática imersiva
